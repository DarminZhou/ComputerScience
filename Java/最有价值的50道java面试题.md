# 最有价值的50道java面试题
## 面向对象的特征有哪些方面?
1. 抽象
抽象是将一类对象的共同特征总结出来构造类的过程，

包括数据抽象和行为抽象

抽象只关注对象有哪些行为和属性，并不关心这些行为的细节是什么

2. 继承
```text
继承是从已有类得到继承信息创建新类的过程。
提供继承信息的称为父类
得到继承信息的称为子类
```
